# Profil Mahasiswa

## Deskripsi Umum Website

Website Profil Mahasiswa ini merupakan website portofolio statis yang dibangun menggunakan HTML murni (pure HTML) tanpa bantuan CSS eksternal, JavaScript, framework, maupun library tambahan. Website ini dirancang untuk menampilkan identitas, profil, sertifikat, proyek, galeri multimedia, serta informasi kontak pemilik secara terstruktur, rapi, dan mudah dipahami. Fokus utama dari pengembangan website ini adalah pada pemahaman dasar struktur HTML, semantik elemen, serta teknik layouting tradisional menggunakan tabel.

Website ini mengusung tema visual dark mode dengan nuansa modern untuk memberikan kesan profesional, bersih, dan nyaman dilihat. Seluruh tampilan disesuaikan agar tetap informatif dan menarik meskipun hanya menggunakan fitur bawaan HTML.

---

## Struktur dan Konten Website

Website ini terdiri dari beberapa bagian utama yang saling terhubung melalui navigasi internal (anchor link).

Bagian **Hero Section** berfungsi sebagai tampilan pembuka yang menampilkan identitas singkat pemilik website, peran atau bidang keahlian, serta tombol call-to-action menuju proyek dan media profesional. Bagian ini dirancang untuk memberikan kesan pertama yang jelas dan informatif kepada pengunjung.

Bagian **About Me** berisi penjelasan singkat mengenai latar belakang akademik dan minat pemilik website di bidang teknologi, khususnya informatika dan pengembangan web. Informasi ini dilengkapi dengan visual pendukung berupa foto dan label peran untuk memperkuat identitas dan kredibilitas.

Bagian **Sertifikat** menampilkan pencapaian akademik dan non-akademik dalam bentuk kartu sertifikat. Setiap sertifikat dapat diklik untuk melihat versi ukuran penuh, sehingga pengunjung dapat meninjau bukti kompetensi yang dimiliki secara langsung.

Bagian **Project** berisi daftar proyek yang pernah dikerjakan, lengkap dengan deskripsi singkat dan tautan langsung menuju website hasil proyek. Setiap proyek disajikan dalam bentuk card agar mudah dibedakan, terstruktur, dan terlihat rapi.

Bagian **Gallery** digunakan untuk menampilkan konten multimedia berupa video lokal dan video YouTube yang di-embed. Bagian ini menunjukkan bahwa HTML mampu mengakomodasi berbagai jenis media tanpa memerlukan teknologi tambahan.

Bagian **Footer** berfungsi sebagai penutup website yang berisi ringkasan identitas, navigasi cepat, informasi kontak, formulir berlangganan, serta statistik singkat. Footer dirancang cukup informatif agar pengunjung tetap dapat mengakses informasi penting tanpa harus kembali ke bagian atas halaman.

## Teknologi dan Sintaks yang Digunakan

Website ini sepenuhnya menggunakan HTML sebagai bahasa markup utama. Beberapa elemen HTML yang digunakan antara lain `<table>` sebagai alat utama layouting, `<tr>` dan `<td>` untuk pengaturan baris dan kolom, `<font>` untuk pengaturan teks, `<img>` untuk menampilkan gambar, `<video>` dan `<iframe>` untuk menampilkan konten multimedia, serta `<a>` untuk navigasi internal dan tautan eksternal.

Penggunaan SVG (`<svg>`) dimanfaatkan untuk menciptakan efek transisi gradasi antar section agar tampilan tidak terkesan kaku meskipun tanpa CSS. Selain itu, atribut HTML seperti `bgcolor`, `align`, `width`, `cellpadding`, `valign`, dan `target` digunakan untuk mengatur tata letak dan tampilan visual halaman.

Pemilihan layout berbasis tabel dilakukan karena menyesuaikan dengan ketentuan penggunaan HTML murni serta sebagai latihan memahami teknik layouting klasik sebelum menggunakan teknologi layout modern seperti Flexbox atau Grid.

---

## Kesimpulan

Website Profil Mahasiswa ini menunjukkan bahwa dengan pemahaman dasar HTML yang baik, sebuah website yang informatif, terstruktur, dan layak secara visual tetap dapat dibangun tanpa bantuan teknologi tambahan. Proyek ini berfungsi tidak hanya sebagai portofolio digital, tetapi juga sebagai sarana pembelajaran untuk memahami konsep fundamental pengembangan web.
